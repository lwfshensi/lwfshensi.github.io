如何搭建新版本hexo

我先是在debian8.6上用官网的方式搭建的，失败～
然后用aptget 的方式安装nodejs 再装hexo还是失败～
最后我开了一个ubuntu14.04的vm，在里面执行
apt-get install nodejs npm
apt-get install nodejs-legacy
sudo npm install -g hexo-cli
